# Manual de usuario

## 1. Instalacion del APK

1. Copiar el archivo `app-release.apk` al dispositivo Android.
2. Abrir el archivo desde el administrador de archivos.
3. Aceptar la instalacion desde origen externo si Android lo solicita.
4. Abrir la aplicacion OCR DNI.

## 2. Permisos

La aplicacion puede solicitar:

- Camara: para capturar el DNI.
- Imagenes o archivos: para seleccionar una imagen desde galeria.

## 3. Flujo principal

1. Seleccionar sede.
2. Capturar DNI con la camara o elegir una imagen.
3. Procesar OCR.
4. Revisar los datos detectados.
5. Corregir manualmente si corresponde.
6. Guardar el registro.
7. Exportar padron en Excel.

## 4. Recomendaciones de captura

- Usar buena iluminacion.
- Evitar reflejos sobre el DNI.
- Mantener el documento completo dentro del encuadre.
- Evitar imagenes borrosas o inclinadas.

## 5. Limitaciones

- La precision depende de la calidad de imagen.
- OCR puede requerir correccion manual.
- La aplicacion esta orientada a uso offline en Android.
